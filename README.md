# Project_inj
#  Проект: Анализ продаж Walmart с использованием PySpark

## Цель проекта
Цель проекта — проанализировать данные о продажах Walmart, выявить заказы с высокой суммой (`amount > 100`), а также изучить структуру и статистику данных.

## Используемые инструменты
- Python 3.11
- Apache Spark (PySpark)
- CSV формат данных
